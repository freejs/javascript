## Object

Object 라는 객체.  
어떠한 데이터를 저장하는 배열같은 Container(그릇)라고 할 수 있다.  

ultra 는 (명시하지 않아도)Object 라는 객체를 상속받는다.  
즉, Object 라는 객체는 모든 부모의 객체가 되기 때문에 모든 객체들의 prototype(원형)이 된다. Object가 갖고 있는 prototype은 모든 객체가 공통적으로 가지고 있는 기능이 된다.
